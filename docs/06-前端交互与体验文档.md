## 一、视觉风格定义：现代毛玻璃 + 多巴胺配色

### 1.1 设计理念

**Glassmorphism × Dopamine** — 毛玻璃质感的 UI 组件搭配高饱和度多巴胺色彩，参考"蚂蚁阿福"风格。大圆角卡片、柔和阴影、微透明层次感，营造现代、温暖、专业的视觉体验。核心关键词：**教育紫主调、毛玻璃层次、3D 虚拟形象、对话即一切**。

## 二、对话页布局（参考蚂蚁阿福）

### 2.1 整体布局结构

"首页即对话"——顶部是导航栏（品牌标题 + 胶囊按钮），中间是对话消息流（AI气泡+用户气泡+嵌入卡片），底部是输入栏。

```
┌──────────────────────────────────────┐
│  [←]    星途AI          [(···)│(×)]  │  ← 导航栏（毛玻璃背景）
│──────────────────────────────────────│     左：返回按钮 中：品牌渐变标题
│                                      │     右：胶囊按钮（三点菜单+关闭）
│  🤖 早上好，乐乐妈妈！              │
│     乐乐今天状态怎么样？            │  ← AI 气泡（白色毛玻璃背景）
│                                      │
│                    今天练什么？ 👩   │  ← 用户气泡（紫色渐变背景，白字）
│                                      │
│  🤖 今天安排了3个活动...            │
│     ┌─────────────────────────┐     │
│     │ 📋 训练安排卡片          │     │  ← 嵌入卡片（毛玻璃圆角）
│     │ [查看详情] [开始训练]    │     │
│     └─────────────────────────┘     │
│                                      │
│──────────────────────────────────────│
│  [输入消息...              ]  [↑]   │  ← 输入栏：圆角20px输入框 + 渐变发送按钮
└──────────────────────────────────────┘
```

**右上角胶囊按钮规范：**

小程序风格胶囊按钮，椭圆圆角容器，左侧三点菜单按钮 + 竖线分隔 + 右侧关闭按钮。

| 属性 | 规范 |
|------|------|
| 高度 | 32px |
| 圆角 | 16px（完全胶囊形） |
| 边框 | 1px `rgba(0,0,0,0.08)` |
| 背景 | 毛玻璃 `rgba(255,255,255,0.6)` + `backdrop-filter: blur(10px)` |
| 分隔线 | 竖线 1px `rgba(0,0,0,0.08)`，高度 18px，居中 |
| 左侧"···"按钮 | 菜单入口，三个垂直圆点图标，点击弹出设置页 |
| 右侧"×"按钮 | 关闭/返回，X 图标，点击返回上一页 |
| 按钮热区 | 每侧 40×32px |

### 2.2 AI 虚拟形象

3D/2.5D 卡通星星角色（NanoBanana 或类似工具生成），展示在对话页顶部区域，作为 AI 人格化代表。

| 属性 | 规范 |
|------|------|
| 尺寸 | 120x120px |
| 风格 | 3D 卡通渲染，柔和光影，圆润造型 |
| 底座 | 无底座，悬浮在问候语上方 |
| 微动效 | 常驻呼吸动效（缓慢上下浮动 2px，周期 3s） |

| 状态 | 描述 | 使用场景 |
|------|------|---------|
| idle | 默认悬浮，微微呼吸 + 偶尔眨眼 | 对话页默认状态 |
| wave | 挥手问好，身体微倾 | 每日首次打开 |
| think | 头顶浮现半透明思考气泡（...） | 等待 AI 回复 |
| happy | 跳跃 + 眼睛变成弯月形 | 好消息、训练完成 |
| celebrate | 周围绽放彩色粒子 + 开心跳跃 | 重大里程碑 |
| sad | 微微低头，眼睛下垂 | 不好的消息 |
| guide | 一只手指向右侧 | 引导操作 |

### 2.3 顶部问候区规则

| 时段 | 问候语 | 引导语 |
|------|--------|--------|
| 6:00-11:00 | "早上好，{昵称}！" | "{孩子名}今天状态怎么样？" |
| 11:00-14:00 | "中午好，{昵称}！" | "午休时间，记录一下上午的情况吧" |
| 14:00-18:00 | "下午好，{昵称}！" | "下午的训练安排好了吗？" |
| 18:00-22:00 | "晚上好，{昵称}！" | "今天辛苦了，记录一下今天的收获吧" |
| 22:00-6:00 | "夜深了，{昵称}" | "早点休息，明天继续加油" |

### 2.4 数据概览卡片（顶部嵌入）

位于问候语下方，毛玻璃圆角卡片，展示当前最关键的 2-3 个数据指标。

```
┌─────────────────────────────────────────┐
│  🔥 连续训练 7天        本周完成 4/5    │
│  ████████████░░░░ 80%                   │  ← 圆角渐变进度条
│  📈 社交能力 +12%       😊 情绪稳定 5/7 │
└─────────────────────────────────────────┘
```

- 圆角：16px
- 背景：`rgba(255,255,255,0.85)` + `backdrop-filter: blur(20px)`
- 阴影：Level 1
- 点击整张卡片 → 展开为完整 SummaryCard

### 2.5 推荐操作入口

圆角条状入口（非传统气泡），2x2 网格排列，每个入口：

- 圆角：12px
- 背景：白色半透明 `rgba(255,255,255,0.7)` + 毛玻璃
- 左侧：现代线性图标（24px）
- 文字：14px Medium
- 点击效果：弹性缩放 0.95 → 1.0（spring animation）
- 点击后：发送对应的预设消息到对话流

## 三、卡片通用规范

### 3.1 卡片结构模板

所有卡片遵循统一的结构模板：

```
┌─────────────────────────────────┐  ← 圆角 16px + 毛玻璃背景
│ [图标] 标题          [时间范围] │  ← 头部区域 44px
│─────────────────────────────────│  ← 1px 分割线 rgba(0,0,0,0.06)
│                                 │
│         内容区域                │  ← 自适应高度，最大 280px
│                                 │
│─────────────────────────────────│
│ 📍 点击查看详情            ▲  │  ← 底部提示 32px（辅色文字）
└─────────────────────────────────┘
```

### 3.2 卡片尺寸规范

| 属性 | 缩略态 | 全屏态 |
|------|--------|--------|
| 宽度 | 对话气泡宽度（屏幕宽 - 80px） | 屏幕宽度 |
| 最小高度 | 120px | 屏幕高度 × 0.7 |
| 最大高度 | 280px（超出截断） | 屏幕高度 × 0.95 |
| 圆角 | 16px | 顶部 20px 圆角（Sheet 风格） |
| 边框 | 无边框，依靠阴影和毛玻璃层次 | 无边框，顶部拖拽条（4x40px 圆角条） |
| 内边距 | 16px | 20px |
| 背景 | `rgba(255,255,255,0.85)` + `backdrop-filter: blur(20px)` | `#FFFFFF` |
| 阴影 | `0 2px 8px rgba(0,0,0,0.06)` | `0 -4px 20px rgba(0,0,0,0.1)` |

### 3.3 卡片状态机

每张卡片有 5 种状态：

```
loading → loaded → expanded
                 → error
         → empty
```

| 状态 | 缩略态表现 | 全屏态表现 |
|------|-----------|-----------|
| loading | 骨架屏（渐变闪烁，浅灰→白色循环） | 中心加载动画（星星旋转 + 脉冲光圈） |
| loaded | 正常展示数据 | 正常展示数据 |
| expanded | — | 全屏面板展开 |
| empty | "暂无数据" + 现代插画风空状态 + 引导操作按钮 | 同缩略态放大 |
| error | "加载失败，点击重试" + 现代插画风错误状态 | 同缩略态放大 + 重试按钮 |

## 四、卡片详细规范

> 本章定义 23 种卡片的视觉规范，完整卡片类型清单见 `05-前端设计文档.md` §2.4。

### 4.1 SummaryCard — 数据概览卡片

**用途：** 展示一组关键指标的汇总数据，支持同比/环比对比。

**数据源：** 跨表聚合查询（training_sessions、behavior_records、life_records 等）

**缩略态布局：**
```
┌─────────────────────────────────┐
│ 📊 本周概览              近7天  │
│─────────────────────────────────│
│  🏋️ 训练完成    📈 能力提升    │
│    12/15       +8.5%          │
│    ▲ 80%       ▲ 较上周+2%    │
│                                │
│  😊 情绪稳定    🎯 目标达标    │
│    4/7天        68%           │
│    ▼ 较上周-1天  ▲ +6%        │
│─────────────────────────────────│
│ 📍 点击查看详情            ▲  │
└─────────────────────────────────┘
```

**字段规范：**

| 字段 | 类型 | 来源 | 缩略态 | 全屏态 |
|------|------|------|--------|--------|
| metrics | array | data_source.metrics | 最多展示 4 个指标（2x2 网格） | 展示全部指标 + 详细趋势 |
| metric.label | string | 配置 | 指标名称 + 线性图标 | 同左 |
| metric.value | number/string | 查询结果 | 大号数字（DIN 字体 24px） | 同左 |
| metric.unit | string | 配置 | 单位（%、天、次等） | 同左 |
| metric.trend | enum | 计算 | ▲/▼ 箭头 + 百分比，绿涨红跌 | 箭头 + 百分比 + 迷你折线图 |
| metric.comparison_label | string | 配置 | "较上周" / "较上月" | 同左 |
| time_range | object | Agent 传入 | 右上角显示 "近7天" | 时间选择器可切换 |

**全屏态额外内容：**
- 每个指标下方展示迷你趋势折线图（近 N 天）
- 底部 AI 洞察条："本周训练完成率提升，建议保持当前节奏"
- 支持点击单个指标跳转到对应的详细卡片

```
         ┌──────────────────┐
         │                  │
         │   [图片缩略图]    │  ← 最大宽度 65%，圆角 12px
         │   240×180px      │     box-shadow: 0 2px 8px rgba(0,0,0,0.1)
```

| 属性 | 规范 |

| 圆角 | 12px |
| 阴影 | `0 2px 8px rgba(0,0,0,0.1)` |
| 加载态 | 灰色骨架屏 + 渐变闪烁 |
| 失败态 | 灰色背景 + 断裂图片图标 + "点击重试" |

1. 用户发送图片 → 上传到 Supabase Storage
2. 图片 URL 随消息发送到 orchestrator

4. AI 回复中可引用图片内容

- 按钮间距：8px
- 点击效果：弹性缩放 0.95 → 1.0（spring animation，200ms）

**操作按钮触发规则：**

|---------|---------|

#### 5.1.5 图表在对话中的展示规范

**用途：** AI 回复中嵌入数据图表（趋势曲线、雷达图、柱状图等），以卡片缩略态形式展示。

**渲染方式：**
- 缩略态：SVG 内嵌渲染（轻量、快速、跟随主题色）
- 全屏态：victory-native / react-native-chart-kit 渲染（支持交互）

**图表配色规则：**
- 主数据线/面积：角色主色
- 对比数据线：角色主色 40% 透明度
- 网格线：`rgba(0,0,0,0.06)`
- 坐标轴标签：`#9CA3AF`
- 数据点：角色主色实心圆，r=3（缩略态）/ r=5（全屏态）

**图表类型缩略态尺寸：**

| 图表类型 | 缩略态高度 | 全屏态高度 |
|---------|-----------|-----------|
| TrendChart（折线） | 80px | 240px |
| BarChart（柱状） | 80px | 240px |
| RadarChart（雷达） | 160px | 280px |
| HeatmapCard（热力） | 120px | 320px |
| SleepChart（睡眠） | 80px | 200px |

**交互：**
- 缩略态：仅展示，点击整张卡片展开全屏
- 全屏态：双指缩放、左右滑动切换时间段、点击数据点显示详情 tooltip

#### 5.1.6 AI 气泡内表格渲染规范

**用途：** AI 回复中嵌入 HTML 表格，用于数据对比、评估结果、训练统计等结构化信息展示。

**渲染规则：**

| 属性 | 规范 |
|------|------|
| 表头背景 | 角色主色浅底 `rgba(主色, 0.08)` |
| 表头文字 | 角色主色，13px Medium |
| 表头底部 | 1.5px 分隔线 `rgba(主色, 0.2)` |
| 单元格 padding | 5-6px |
| 单元格底部 | 1px 分隔线 `rgba(0,0,0,0.06)` |
| 正文字号 | 12px Regular |
| 表格圆角 | 8px（跟随气泡内嵌，overflow: hidden） |
| 最大宽度 | 100%（撑满气泡宽度） |

**数据对比标记：**

| 趋势 | 标记 | 颜色 |
|------|------|------|
| 上升/进步 | ↑ | `#4CAF50`（绿） |
| 下降/退步 | ↓ | `#F44336`（红） |
| 持平/不变 | → | `#9E9E9E`（灰） |

**示例：**
```
┌──────────┬────────┬────────┬──────┐
│ 领域     │ 上月   │ 本月   │ 变化 │  ← 表头：主色浅底
├──────────┼────────┼────────┼──────┤
│ 社交     │ 35     │ 42     │ ↑+20%│  ← 绿色
│ 沟通     │ 28     │ 30     │ ↑+7% │  ← 绿色
│ 认知     │ 45     │ 45     │ →持平│  ← 灰色
│ 自理     │ 52     │ 48     │ ↓-8% │  ← 红色
└──────────┴────────┴────────┴──────┘
```

**边界情况：**
- 列数 > 5：表格支持水平滚动，固定首列
- 行数 > 10：缩略态最多显示 5 行 + "查看全部 N 行"，全屏态显示全部
- 单元格内容过长：单行截断 + 省略号，点击展开

## 九、原型 v2 新增页面交互规范

### 9.1 训练周报页（10-training-weekly）

对话中用户请求"给我一周训练记录"，AI生成周报卡片嵌入对话流。

**周报卡片结构：**
- 头部：📋 图标 + "一周训练记录" + 日期范围 + 完成率徽章
- 统计行：3列网格（训练天数 / 完成项目 / 平均评分），数字使用渐变色
- 每日条目：左侧状态圆点（绿=优秀/黄=一般/红=较差/灰=休息）+ 日期 + 训练项目标签（flex-wrap自动换行）+ 右侧评分
- 技能进度：六大领域进度条（语言沟通/社交互动/认知学习/感觉运动/情绪行为/生活适应），渐变填充
- 底部操作：[查看详细记录]（primary）+ [分享给老师]（secondary）

**训练项目标签样式：**
- 胶囊形 `padding: 2px 8px; border-radius: 6px`
- 背景 `rgba(124,77,255,0.06)`，文字 `#6B7280`
- flex-wrap 自动换行，支持每天超过3个训练项目

### 9.2 分析报告页（11-analysis-report）

对话中用户请求"给我一份分析报告"，AI生成综合分析报告卡片。

**报告卡片结构：**
- 头部：📊 图标 + "综合发展分析" + 评估周期 + 评估等级徽章
- 六大领域雷达图：SVG绘制，六个领域（语言沟通/社交互动/认知学习/感觉运动/情绪行为/生活适应），渐变填充
- 维度评分：每个维度的进度条 + 分数 + 等级标签
- 趋势对比：3个季度的柱状对比（Q4 2025 / Q1 2026 / 本季度）
- ABC行为分析：
  - 统计概览：总记录数 + 已识别模式 + 干预有效率
  - 行为模式卡片（每个行为一张）：行为名称+频率+等级 → A(前因)/B(行为)/C(后果)三行带彩色标签 → AI洞察
  - 标签颜色：A=橙色(#FF9800) B=红色(#EF4444) C=蓝色(#2196F3)
- 关键发现：带图标的发现条目
- 底部操作：[下载PDF] + [分享给医生]

### 9.3 训练详情页（12-training-detail）

从周报卡片"查看详细记录"按钮进入，超长滚动页面。

**页面结构：**
- 顶部导航栏：返回按钮 + "训练详情" + 胶囊按钮
- 周切换器：左右箭头 + 周标签（本周/上周/上上周），支持3周数据
- 周统计概览：4列网格（训练天数/完成项目/平均评分/总时长）
- 日期锚点导航栏：横向滚动的日期按钮，点击平滑滚动到对应日期区域，滚动时自动高亮当前日期
- 每日区域：
  - 粘性日期头（sticky header）：日期 + 训练项目数
  - 训练课程卡片：课程名称 + 目标 + 进度条 + 训练方法 + 标签
  - 部分日期显示AI总结而非详细卡片

**交互特点：**
- 所有天数垂直排列，不隐藏
- 日期导航栏固定在周统计下方
- 滚动监听自动高亮当前可见日期
- 周切换时重新渲染所有数据

### 9.4 语音记录页（09-voice-record）

对话中长按录音按钮，语音描述训练过程，AI自动结构化。

**交互流程：**
1. 用户长按录音按钮 → 显示录音波形动画
2. 松开结束录音 → 语音气泡出现在对话流
3. AI处理中 → 打字动画
4. AI返回结构化训练记录卡片，包含：
   - 训练目标（自动提取）
   - 完成度（百分比进度条）
   - 训练方法（标签）
   - 观察笔记（AI整理的文字）
   - 操作按钮：[确认记录] [修改内容]

### 9.5 设置页（04-settings）

从对话页右上角胶囊菜单进入，右滑进入动画。

**主列表布局：**
- 用户信息头部：头像 + 昵称 + 手机号
- 功能分组（毛玻璃卡片容器）：
  - 账号信息：修改昵称、修改手机号
  - 偏好设置：使用偏好、数据访问权限
  - 法律信息：用户协议、隐私政策
  - 其他：关于星途、意见反馈
- 底部操作：升级VIP（渐变按钮）、注销账号（红色文字）

**子页面交互：**
- 从右侧 slide-in 进入（translateX(100%) → translateX(0)）
- 顶部有返回按钮 + 子页面标题
- 数据访问权限子页面：顶部孩子切换Tab（多孩家庭），每个孩子独立的权限面板
- 用户协议/隐私政策：长文本滚动页面，内容与登录页一致
