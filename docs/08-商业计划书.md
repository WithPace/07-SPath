## 一、执行摘要

> 星途AI是中国首个纯AI驱动的自闭症儿童康复服务平台，用一个对话窗口替代传统康复管理软件的全部功能。

**核心数据**

| 指标 | 数据 |
|------|------|
| 目标市场规模（TAM） | ¥1,500亿/年（300万家庭 × 年均康复支出¥5万） |
| 可服务市场（SAM） | ¥45亿/年（数字化康复管理工具市场） |
| 可获取市场（SOM，3年目标） | ¥4,800万/年（10万付费家庭 × ARPU ¥40/月） |
| 产品形态 | iOS + Android App + 微信小程序 |
| 商业模式 | C端订阅（¥29-59/月）+ B端机构（¥499/月）+ 电商带货（训练教具/健康设备/穿戴设备）+ 增值服务 |
| MVP进度 | Phase 1 开发中，预计24周完成全部三阶段 |
| 团队 | 创始人全栈开发，计划组建5人核心团队 |

**融资需求**：种子轮融资 ¥500万，出让10-15%股权，用于产品开发、团队组建和早期获客。

**一句话定位**：星途AI = 自闭症康复领域的"AI私人顾问"，让每个家庭都能获得专业级康复指导。

## 二、市场机会

> 中国自闭症康复市场是一个千亿级刚需市场，供需严重失衡，数字化渗透率极低。

### 2.1 市场规模

```
TAM（总可寻址市场）
├── 300万自闭症家庭 × ¥5-15万/年康复支出 = ¥1,500-4,500亿/年
│
SAM（可服务市场）
├── 数字化康复管理工具 = ¥45亿/年
├── 计算依据：300万家庭 × 15%数字化渗透率 × ¥1,000/年工具支出
│
SOM（3年可获取市场）
├── 10万付费家庭 × ¥1,200/年 = ¥1.2亿/年
├── 5,000家机构 × ¥6,000/年 = ¥0.3亿/年
└── 合计 ¥1.5亿/年
```

### 2.2 行业痛点（数据驱动）

| 痛点 | 数据支撑 |
|------|---------|
| 专业资源极度稀缺 | 全国仅约5万名相关医生，300万家庭对应60:1的供需比 |
| 康复费用高昂 | 家庭年均支出5-15万元，70%以上家庭感到经济压力沉重 |
| 居家训练无指导 | 机构训练每周仅10-20小时，剩余148小时家长不知如何干预 |
| 数据严重割裂 | 家长、医生、特教老师三方信息靠微信群传递，无统一数据平台 |
| 记录难以坚持 | 传统纸质/表单记录方式，家长坚持率不足20% |
| 评估周期长 | 从疑似到确诊平均等待3-6个月，错过早期干预黄金期 |

### 2.3 市场趋势

**政策利好**

- 2025年《"十四五"残疾人保障和发展规划》持续推进，自闭症康复纳入基本公共服务
- 多地将自闭症康复训练纳入医保报销范围，家庭支付能力提升
- 《儿童个人信息网络保护规定》完善，为数字化康复产品提供合规框架

**技术成熟**

- 大语言模型中文能力大幅提升，Doubao/Kimi等国产模型成本持续下降
- Serverless架构（Supabase Edge Functions）使创业团队能以极低成本构建企业级后端
- 流式对话（Streamable HTTP）技术成熟，AI交互体验接近真人对话

**用户意识提升**

- 自闭症早期筛查意识增强，每年新增确诊约16万例
- 80后90后家长数字化素养高，愿意尝试AI辅助工具
- 家长社群（微信群、小红书、抖音）活跃，口碑传播效率高

## 三、产品与解决方案

> 星途AI的核心差异化：AI不是辅助工具，AI就是产品本身。用户只需要和一个"懂孩子的AI伙伴"对话，所有康复管理工作自动完成。

### 3.1 核心差异化

| 维度 | 传统康复管理软件 | 星途AI |
|------|----------------|--------|
| 交互方式 | 表单填写、页面跳转、按钮操作 | 对话即一切，说话就能完成所有操作 |
| 智能程度 | AI作为辅助功能（生成报告） | AI作为唯一服务入口，自主编排所有能力 |
| 个性化 | 基于规则的推荐 | 基于长期记忆的深度个性化（记住每个孩子的偏好、策略、触发因素） |
| 数据连续性 | 手动录入，数据碎片化 | 对话自动结构化，数据持续积累形成完整画像 |
| 多角色协作 | 各用各的系统 | 家长-医生-老师在同一平台，AI自动翻译和同步信息 |
| 使用门槛 | 需要学习复杂操作 | 会说话就会用 |

### 3.2 产品架构

```
用户（家长/医生/老师/机构）
        │
        ▼
   AI 对话窗口 + 动态看板
        │
        ▼
   Orchestrator（意图识别，Doubao-32k-pro）
        │
   ┌────┼────┬────┬────┬────┬────┐
   ▼    ▼    ▼    ▼    ▼    ▼    ▼
 日常  评估  训练  行为  生活  看板  协作
 对话  筛查  管理  危机  记录  分析  通知
   │    │    │    │    │    │    │
   └────┴────┴────┴────┴────┴────┘
                  │
                  ▼
        Supabase（PostgreSQL + RLS）
        双层快照（short_term / long_term）
```

### 3.3 用户价值主张

| 角色 | 核心价值 | 关键场景 |
|------|---------|---------|
| 家长 | "一个24小时在线、记得我孩子所有情况的康复顾问" | 居家训练指导、M-CHAT筛查、情绪支持、行为危机应对、训练记录（语音即可） |
| 医生 | "一个帮我追踪患儿居家数据的智能助手" | 连续数据查看、评估辅助、远程建议、协作报告 |
| 特教老师 | "一个帮我快速出IEP、自动记录教学数据的搭档" | IEP生成、课堂记录、家校信息同步、个性化教学策略 |
| 机构管理 | "一个帮我管理所有学员数据和教师绩效的运营平台" | 学员概览、教师管理、运营数据、批量报告 |

### 3.4 产品亮点

- **对话即服务**：无需学习任何操作，说话就能完成评估、记录、分析
- **AI长期记忆**：记住每个孩子的性格、偏好、有效策略、里程碑，越用越懂孩子
- **六领域能力画像**：语言沟通、社交互动、认知学习、感觉运动、情绪行为、生活适应，持续追踪
- **多角色AI人格**：家长端温暖共情、医生端专业严谨、老师端实操导向
- **行为危机实时响应**：孩子出现危机行为时，AI即时提供专业应对指导（Kimi 2.5，不降级）
- **18种数据卡片**：训练卡、评估卡、行为卡、看板卡等，缩略态嵌入对话，点击全屏展开

**电商带货模式说明**：

| 维度 | 说明 |
|------|------|

| 启动时间 | Phase 3 启动CPS分销，Phase 4 启动自营精选 |

### 4.2 定价策略

| 版本 | 月费 | 年费（8折） | 核心权益 | 目标用户 |
|------|------|-----------|---------|---------|
| 免费版 | ¥0 | — | 10条AI对话/天，1次评估/月，基础看板 | 新用户体验，建立信任 |
| 基础版 | ¥29 | ¥278 | 100条/天，3次评估/月，完整看板，语音输入 | 轻度使用家庭 |
| 专业版 | ¥59 | ¥566 | 无限对话+评估，FBA行为分析，周期报告，多模态 | 重度使用家庭 |

- 免费版提供足够价值让用户体验AI能力，降低决策门槛

- 对比线下康复¥5-15万/年，¥59/月性价比极高
- 机构版按机构收费而非按学员收费，降低机构采购决策难度

## 五、竞争分析

> 当前市场以传统表单工具和信息平台为主，尚无真正的"AI原生"康复管理产品。

### 5.1 竞品对比矩阵

| 维度 | 星途AI | 大米和小米App | 恩启IDEA | 小星星在家练 | ALSOLIFE | 北大医疗脑健康 |
|------|--------|-------------|----------|------------|----------|-------------|
| 产品定位 | AI原生康复服务平台 | 康复机构管理+家长社区 | 特教评估与课程工具 | 居家训练视频课程 | 评估+数据追踪平台 | 线上线下康复服务 |
| 交互方式 | AI对话（唯一入口） | 传统App页面 | 传统Web/App | 视频课程+打卡 | 表单+图表 | 传统App+线下 |
| AI能力 | 全链路AI驱动 | 无/弱AI | 评估辅助 | 无 | 评估算法 | 弱AI辅助 |
| 个性化程度 | 深度个性化（AI记忆） | 低（通用内容） | 中（评估驱动） | 低（标准课程） | 中（数据驱动） | 中（人工定制） |

| 数据连续性 | 自动积累，AI结构化 | 手动录入 | 手动录入 | 无数据追踪 | 手动+半自动 | 手动录入 |
| 月费 | ¥0-99（C端） | 线下课程为主 | ¥100-300 | ¥50-200 | ¥0-199 | 线下课程为主 |
| 核心壁垒 | AI记忆+数据飞轮 | 线下机构网络 | 评估量表版权 | 课程内容 | 评估数据库 | 医疗资质+线下网点 |

### 5.2 竞争壁垒

**第一层：AI记忆壁垒（短期）**

星途AI为每个孩子构建独立的"数据域"，包含双层语义快照（short_term 30天滑动窗口 + long_term 完整发展档案）、AI长期记忆（性格、偏好、触发因素、有效策略）、六领域能力画像历史。用户使用越久，AI越懂孩子，迁移成本越高。

**第二层：数据飞轮壁垒（中期）**

```
更多用户使用 → 更多训练/行为/生活数据
       ↓
AI模型持续优化（Prompt调优 + 场景积累）
       ↓
更精准的个性化建议 → 更高用户满意度和留存
       ↓
口碑传播 → 更多用户使用
```

**第三层：多角色网络效应（长期）**

家长邀请医生和老师加入照护团队 → 医生/老师体验产品 → 推荐给其他家长 → 形成"家长-医生-老师"三角网络。每新增一个角色，平台对其他角色的价值都在增加。机构采购进一步放大网络效应。

**第四层：技术架构壁垒**

- Orchestrator + Edge Functions 架构实现了"对话即服务"的产品范式，竞品难以快速复制
- 双层快照机制保证AI在毫秒级获取完整上下文，传统架构需要多次数据库查询
- 四种角色Agent各有独立Prompt和行为约束，专业度远超通用AI套壳产品
